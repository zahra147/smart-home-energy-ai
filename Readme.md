# 🏠 AI-Powered Smart Home Energy Analytics

> Intelligent Energy Consumption Prediction and Decision Support using Machine Learning for Smart Homes and Industrial Energy Management.

---

## 📌 Overview

This project presents an AI-driven approach for analyzing and predicting household energy consumption using Machine Learning techniques.

The objective is not only to forecast energy usage but also to support intelligent decision-making for improving energy efficiency in smart homes.

The project is designed as part of a larger Industrial AI ecosystem that integrates IoT sensor data, predictive analytics, and Human-Machine Interface (HMI) technologies.

---

## 🎯 Project Goals

- Predict household energy consumption
- Discover hidden consumption patterns
- Analyze influential environmental factors
- Support intelligent energy management
- Build a scalable AI pipeline for Smart Home applications

## 📊 Dataset Features

The dataset contains environmental and temporal measurements including:

- Indoor Temperature
- Outdoor Temperature
- Indoor Humidity
- Outdoor Humidity
- Temperature Difference
- Hour of Day
- Historical Energy Consumption

Additional engineered features are generated during preprocessing.

---

## 📈 Results

### Feature Importance Analysis

The model identifies the most influential variables affecting household energy consumption.

Example findings include:

- Hour of Day
- Indoor Temperature
- Indoor Humidity
- Outdoor Humidity
- Temperature Difference

---

### Pattern Discovery

PCA combined with K-Means clustering is used to identify different household consumption behaviors.

These insights can later be utilized for personalized energy optimization strategies.
